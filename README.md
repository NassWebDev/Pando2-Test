Setup
Make sure to install the dependencies:

# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
Development Server
Start the development server on http://localhost:3000:

# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
Production
Build the application for production:

# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
Locally preview production build:

# npm
npm run preview

# pnpm
pnpm run preview

# yarn
yarn preview
